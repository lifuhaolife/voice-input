# 资源管理检查清单

## 每次代码修改必须检查

### 1. CPU 占用检查
- [ ] 循环中是否有 `time.sleep()` 避免空转
- [ ] 轮询超时是否合理（推荐 0.5 秒）
- [ ] 回调函数是否提前返回（不处理时）
- [ ] 主循环休眠时间是否足够（推荐 0.5 秒）

### 2. 内存泄漏检查
- [ ] 列表/队列是否无限增长
- [ ] 线程是否正确清理（daemon=True）
- [ ] 回调是否被重复注册
- [ ] 缓存是否有上限

### 3. 资源释放检查
- [ ] WebSocket 连接是否正确关闭
- [ ] 音频流是否正确关闭
- [ ] 文件句柄是否关闭
- [ ] 线程是否可终止

### 4. 线程安全检查
- [ ] 共享变量是否有锁保护
- [ ] 线程退出标志是否正确设置
- [ ] 是否使用 daemon 线程
